Basis sets

There are basis sets of LCModel model metabolite spectra for 1.5T, 3T, 7T and 9.4T. Other sites have tools for simulating LCModel basis sets, for example http://scion.duhs.duke.edu/vespa/ Suppose you want to install the 3T basis sets: Download 3t.zip (32 Mbyte) 

	   mv  3t.zip  ~/.lcmodel/basis-sets/
           cd   ~/.lcmodel/basis-sets
           unzip  3t.zip
Important: Be sure to read the README file, e.g., in ~/.lcmodel/basis-sets/README-3t.txt

For the other field strengths:
1.5t.zip (25 Mbyte),
7t.zip (19 Mbyte),
9.4t.zip (19 Mbyte).
