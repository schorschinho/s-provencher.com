# s-provencher.com backup

Last scraped: November 23,2025

This repository contains the entire content of the website s-provencher.com.

s-provencher.com is a website developed and maintained by Stephen Provencher, PhD. It contains valuable resources, source code, and manuals for multiple computer programs that are widely used to solve complex inverse problems:
- LCModel
- CONTIN
- DISCRETE
- SplMod

Dr. Provencher [sadly passed away in 2025](https://www.koprivataylor.com/obituary/stephen-provencher-phd). This repository seeks to ensure that his work and legacy will remain accessible for future generations of researchers, regardless of the state of the original s-provencher.com website and domain.
